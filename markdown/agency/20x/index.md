# 20x Overview

FedRAMP 20x is a modernized certification path focused on automation, security outcomes, and reusable security information. Agencies using 20x materials should review the certification class, key security indicators, validation information, and provider-supplied evidence for the intended federal use.

20x is intended to make authorization decisions faster and more repeatable while preserving agency accountability for the systems agencies authorize.
