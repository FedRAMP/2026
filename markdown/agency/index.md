# Agency Guidance


Federal agencies use FedRAMP Certification information to support authorization decisions, manage ongoing cloud risk, and coordinate with FedRAMP and other agencies. CR26 is intended to make those responsibilities clearer by separating government-wide FedRAMP rules from agency-specific authorization decisions.

Agencies should use FedRAMP materials to understand the provider's certification status, the certification class and path, the security information available through the Marketplace, and any additional agency requirements needed for the agency mission or system boundary.
