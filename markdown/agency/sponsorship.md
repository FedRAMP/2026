# Sponsorship

Agency sponsorship can help providers pursue a FedRAMP path that requires agency participation. Sponsorship should be based on an agency's mission need, intended use of the service, and ability to participate in the required authorization and monitoring activities.

Sponsoring agencies should coordinate with the provider, FedRAMP, and independent assessors when applicable so that responsibilities, evidence expectations, and timelines are understood before work begins.
