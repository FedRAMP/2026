# Certification Classes

Certification Classes organize FedRAMP Certifications by the level and kind of assurance required. Agencies should use the certification class to understand the expected security information, assessment depth, and monitoring obligations for a cloud service.

The certification class does not remove agency responsibility for authorization decisions. Agencies should still review whether the service, boundary, data types, and use case fit the agency mission and risk tolerance.
