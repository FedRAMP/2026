# Initial Authorization

Initial authorization is the agency decision to authorize a federal information system that uses a cloud service. FedRAMP Certification information supports that decision by supplying reusable evidence, assessment results, provider commitments, and monitoring expectations.

Agencies should review FedRAMP materials, confirm the service is appropriate for the intended use, document agency-specific risk decisions, and identify any additional agency requirements that apply.
