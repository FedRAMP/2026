# Rev5 Overview

Rev5 remains an important FedRAMP certification path for cloud services that use National Institute of Standards and Technology Special Publication 800-53 control baselines. Agencies using Rev5 materials should focus on the provider's certification status, assessment materials, ongoing certification information, and any agency-specific authorization needs.

CR26 presents Rev5 guidance alongside FedRAMP 20x so agencies can understand which rules apply to each certification path.
