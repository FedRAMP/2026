# Agency-Specific Continuous Monitoring

Agency-specific continuous monitoring covers the additional information an agency needs for its own authorization, mission, system boundary, or operating environment. These activities should complement FedRAMP's government-wide monitoring rules.

Agencies should avoid recreating government-wide monitoring requests when FedRAMP information already supplies what is needed. Agency-specific requests should be scoped to the agency's use case and risk decision.
