# Collaborative Continuous Monitoring

Collaborative Continuous Monitoring rules support shared visibility into the ongoing security posture of FedRAMP-certified cloud services. Agencies should follow the Collaborative Continuous Monitoring rules when using shared monitoring information supplied through FedRAMP channels.

The goal is to reduce duplicative agency requests while improving consistency, timeliness, and reuse of provider-supplied monitoring information.
