# Rev5 Overview

Rev5 certification work uses National Institute of Standards and Technology Special Publication 800-53 control baselines. Providers pursuing Rev5 should prepare control implementation information, assessment materials, plans of action, monitoring evidence, and package information that supports FedRAMP and agency review.

CR26 keeps Rev5 guidance available while aligning terminology and rule structure with the broader Consolidated Rules for 2026.
