# Initial Certification

A Rev5 initial certification is the provider's first full assessment activity for a cloud service under the applicable Rev5 baseline and certification class. Providers should prepare the system boundary, control implementation information, required evidence, assessor access, and submission materials before the assessment begins.

The provider should supply complete, accurate materials and should respond to assessor findings in a way that supports FedRAMP and agency reuse.
