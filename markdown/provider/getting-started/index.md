# Getting Started

Providers should begin by identifying the service boundary, likely impact level, target certification type, certification path, and certification class. Providers should also determine whether they need an advisor, an independent assessor, agency sponsorship, or a Marketplace listing before pursuing certification.

Early planning should focus on evidence quality, automated security information where applicable, clear ownership, and readiness to supply information through FedRAMP channels.
