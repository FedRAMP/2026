# Finding an Advisor

Providers may use advisors to understand FedRAMP expectations, prepare materials, and plan certification work. Advisors can be useful when a provider is new to FedRAMP or needs help aligning internal security practices with CR26 rules.

Using an advisor does not transfer responsibility. Providers remain accountable for the accuracy of supplied information and for compliance with FedRAMP rules.
