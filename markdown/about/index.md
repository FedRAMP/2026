# About CR26 Documentation

The Consolidated Rules for 2026 documentation brings FedRAMP modernization materials together in a single site for agencies, cloud service providers, independent assessors, and advisors.

The documentation combines human-written orientation pages with machine-generated rule pages created from FedRAMP machine-readable rules.
