# Source Data

The Consolidated Rules for 2026 documentation is built from 2 primary source types.

Human-written narrative pages are drafted from FedRAMP modernization materials, program guidance, and stakeholder-specific context. These pages explain how the rules fit together and how agencies, providers, assessors, and advisors should use them.

Machine-generated rule pages will be generated from FedRAMP machine-readable rules. Those pages should preserve the structure, labels, descriptions, purposes, applicability statements, and rule text from the source rule data.

When narrative content and machine-readable rules differ, the machine-readable rules should be reviewed as the source of truth for the generated rule content.
