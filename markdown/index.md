# Consolidated Rules for 2026

The Consolidated Rules for 2026 are the next unified expression of FedRAMP rules for cloud service providers, federal agencies, independent assessors, and advisors. The rules consolidate FedRAMP modernization work into a single public documentation set that explains how cloud services are certified, how agencies rely on those certifications, and how FedRAMP keeps the program aligned across government.

CR26 includes rules for FedRAMP 20x, Rev5, Certification Classes, Marketplace participation, Certification Data Sharing, Collaborative Continuous Monitoring, Vulnerability Detection and Response, Significant Change Notifications, and related program responsibilities.

The goal is to make FedRAMP rules easier to use in day-to-day work. Human-written pages explain context, roles, and practical workflows. Machine-generated pages will be generated from FedRAMP machine-readable rules so that implementation details remain consistent with the source rule data.
