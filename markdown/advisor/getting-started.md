# Getting Started

Advisors should begin by understanding current FedRAMP Certification types, classes, paths, and Marketplace rules. Advisor services should be aligned with the current rules and should avoid outdated terminology or historical approaches that no longer reflect CR26.

Advisors seeking Marketplace listing should prepare clear service descriptions, contact information, relevant experience, and any required attestations or supporting information described by FedRAMP rules.
