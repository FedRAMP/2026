# Shared Responsibility Model

FedRAMP uses a shared responsibility model. FedRAMP, cloud service providers, federal agencies, independent assessors, and advisors each have responsibilities that support trustworthy use of cloud services across government.

FedRAMP defines government-wide rules, maintains the Marketplace, reviews certification information, and coordinates shared practices. Providers implement and operate cloud services, supply required security information, maintain evidence, and participate in ongoing monitoring. Agencies use FedRAMP information to make authorization decisions, manage agency-specific risk, and apply additional requirements when needed.

Independent assessors supply objective assessment and validation services. Advisors may help providers understand FedRAMP expectations and prepare materials, but they do not replace provider accountability or independent assessor judgment.
