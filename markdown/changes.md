# Important Changes in CR26

This page is only relevant to parties that are already participating in the FedRAMP process.

A lot is changing in CR26. This page summarizes those changes.

## Rev5 Changes - Mandatory BIR Adoption

- Minimum Assessment Scope: 2027 January 31 or next annual assessment (whichever is later)
- Significant Change Notification: 2027 January 31
- Collaborative Continuous Monitoring: 2027 March 31
- Vulnerability Detection and Response: 2027 May 31
- Authorization Data Sharing: 2027 July 31
