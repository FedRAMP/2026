# Important Changes in CR26

This page is retained for compatibility with the early site structure. The primary change log for CR26 is now the [Changelog](../changelog/) page, and effective-date details will be generated from FedRAMP machine-readable rules.

CR26 introduces a consolidated rule structure for FedRAMP Certification, Certification Classes, FedRAMP 20x, Rev5 modernization, Certification Data Sharing, Collaborative Continuous Monitoring rules, Significant Change Notification rules, and Vulnerability Detection and Response rules.
