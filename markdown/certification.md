# FedRAMP Certification

FedRAMP Certification is a government-wide security designation for cloud services that meet FedRAMP rules. A certification helps agencies understand whether a cloud service is appropriate for federal use and what responsibilities remain with the agency and provider.

FedRAMP Certifications are organized by type, class, and path. Certification types describe the broad authorization model. Certification Classes describe the risk and assurance level that applies to a cloud service. Certification paths describe how a provider gets to certification and how the certification is maintained over time.

Agencies remain responsible for authorizing their own federal information systems. FedRAMP Certification supplies reusable security information that agencies can use when making authorization decisions, managing ongoing risk, and monitoring provider performance.
